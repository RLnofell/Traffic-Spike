HỆ THỐNG PHÂN TÍCH HIỆU NĂNG (Custom Metrics)

Sử dụng Middleware để đo đạc thời gian xử lý (High Resolution Time) của từng request.

const start = process.hrtime(); res.on('finish', () => { const diff = process.hrtime(start); const durationMs = (diff[0] * 1e9 + diff[1]) / 1e6; // ... lưu log và tính trung bình latency });
BẢO MẬT VÀ TỐI ƯU PRODUCTION
Helmet.js: Bảo mật Header HTTP, chống Clickjacking, XSS.
Compression: Nén Gzip dữ liệu trả về, giảm 70% dung lượng băng thông.
Morgan: Logger chuẩn công nghiệp giúp theo dõi log server thời gian thực.
app.use(helmet()); app.use(compression()); app.use(morgan("dev"));
TỐI ƯU HÓA BẰNG HTTP 304 (Caching)

Express tự động tạo ETag. Nếu dữ liệu không đổi, server trả về mã 304 thay vì gửi lại toàn bộ JSON.

Workflow: Request -> Server (Check ETag) -> 304 Not Modified.
